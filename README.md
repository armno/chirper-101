# Chirper

Learning Laravel & React from [Laravel Bootcamp](https://bootcamp.laravel.com).

## setup

```sh
brew install composer
composer install
cp .env.example .env
```

create databalse

```sh
php artisan migrate
```

## running local server

```sh
php artisan key:generate
php artisan serve
npm install
npm run dev
```

open `http://localhost:8000`.
